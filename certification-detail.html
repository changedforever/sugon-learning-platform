初级认证

海光CPU工程师认证

技能描述

具备海光芯片售前、售后技术支持、系统运维、性能优化、故障诊断等必备的知识和技能

开始学习

课程目标

掌握 CPU 微架构、指令集、缓存层次结构等核心概念。

熟悉 CPU 生态体系，包括操作系统、编译器、工具链等。

掌握 CPU 基准测试方法，能够进行基础性能分析与优化。

了解 CPU 在不同应用场景下的性能优化策略。

掌握 CPU 系统的基础运维技能。

具备 CPU 系统售前技术支持的基础能力。

课程模块

M1

模块一：CPU 微架构及相关技术

  • • IC发展历程及芯片产品化过程
  • • 微架构核心概念
  • • 指令集架构 (ISA)
  • • 流水线结构
  • • 缓存层次结构
  • • CPU 微架构介绍
  • • CPU 安全机制简介
M2

模块二：CPU 生态体系

  • • CPU 兼容性概述
  • • CPU 基础软件生态
  • • CPU 编译器及工具链介绍
  • • 海光产业生态合作组织及申请适配认证介绍
M3

模块三：CPU 基准测试与性能分析

  • • 基准测试概述
  • • CPU 常用基准测试工具
  • • 基准测试结果分析基础
  • • 性能瓶颈定位方法简介
  • • 基准测试性能优化方法介绍
M4

模块四：CPU 应用场景性能优化

  • • 虚拟化场景优化基础
  • • 大数据场景优化基础
  • • 数据库场景优化基础
  • • 网络及流量场景优化基础
  • • 海光安全技术测试指南
M5

模块五：CPU 系统运维

  • • 系统部署与配置基础
  • • 系统监控与告警基础
  • • Linux 系统问题排查指南
  • • 应用场景类问题排查指南
M6

模块六：CPU 技术支持

  • • CPU 产品及解决方案介绍
  • • 客户需求分析与方案设计基础
  • • 技术交流与方案讲解基础

完成所有6个模块的学习内容，掌握核心知识点
